Farsi-English JSON Script

API Documentation

This API is to be used on Farsi and English programme scripts. It will help you to format JSON array of strings to JSON array of objects combining Farsi and English transcripts. And it will convert other file formats to a JSON so they can then be formatted into JSON objects.

This is a free API built in Node JS.

Conversion to JSON Array

Convert Farsi and English scripts from Non-JSON formats to a JSON array.

  • Currently supported formats for conversion: .rtf
  • Endpoint: /convert/array

POST method

In order to get an RTF script converted to JSON, you will need to use a POST method on the endpoint.

const rtfFile = "
\b \cf6 OOV:
\b0 Yes
\
\pard\pardeftab720\ri0\sl276\slmult1\rtlpar\qr\partightenfactor0
\cf6 \uc0\u1576 \u1604 \u1607 \
\

\b \cf6 DAVE: \'93Yes\'94\
\
\pard\pardeftab720\ri0\sl276\slmult1\rtlpar\qr\partightenfactor0
\cf6 \uc0\u1576 \u1604 \u1607 \
\"

POST http://localhost:8080/convert/array

input = {format: "rtf", file: rtfFile}

output = ["OOV:", "Yes", "بله", "DAVE:", " 'Yes'", "بله"]

Conversion to JSON Object

Convert Farsi and English scripts from Non-JSON formats to a JSON object.

  • Currently supported formats for conversion: .rtf
  • Endpoint: /convert/object

POST method

In order to get an RTF script converted to JSON, you will need to use a POST method on the endpoint.

const rtfFile = "
\b \cf6 OOV:
\b0 Yes
\
\pard\pardeftab720\ri0\sl276\slmult1\rtlpar\qr\partightenfactor0
\cf6 \uc0\u1576 \u1604 \u1607 \
\

\b \cf6 DAVE: \'93Yes\'94\
\
\pard\pardeftab720\ri0\sl276\slmult1\rtlpar\qr\partightenfactor0
\cf6 \uc0\u1576 \u1604 \u1607 \
\"

POST http://localhost:8080/convert/object

input = {format: "rtf", file: rtfFile}

output = {
  "title": "Script",
  "scripts": [
    {"meta": "OOV", "english": "Yes", "farsi": "بله"},
    {"meta": "DAVE", "english": " 'Yes'","farsi": "بله"}
  ]
}

Format

Format Farsi and English scripts from an JSON array of strings to a JSON object.

  • Endpoint: /format

POST method

This endpoint will convert your JSON array of string into a JSON object putting in parallele both languages.

POST http://localhost:8080/format

input = {file: ["OOV:", "Yes", "بله", "PIX- Dave using the app","DAVE: 'Yes'", "بله"]}

output = {
  "title": "Script",
  "scripts": [
    {"meta": "OOV", "english": "Yes", "farsi": "بله"},
    {meta: "PIX- Dave using the app"},
    {"meta": "DAVE", "english": "Yes","farsi": "بله"}
  ]
}
